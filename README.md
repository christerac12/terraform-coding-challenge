# terraform-coding-challenge
Terraform Lab for University of Utah interview process. 



Goals:
Use the main.tf file and modularize it following best practices. (the ami can be updated to ami id ubuntu 22.04 for that region). Check for any other errors in the code and solve them as well.

Test out the modified code by testing out internet connectivity of the ec2 instance in private subnet.

Push the modified code to the Github repo and ping us via e-mail for code review.
